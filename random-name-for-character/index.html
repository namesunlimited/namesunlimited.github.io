    About this list

    Built for writers and world-builders

    Character names in this generator are built from real first and last names, then combined at random — so results read as plausible, grounded names rather than obviously fake fantasy words.

    Generate a batch, skim for one that fits your character's voice, and regenerate as many times as you like until something clicks.

    A good character name usually does one of three jobs: it signals background (era, culture, class), it's easy for a reader to say in their head, or it simply gets out of the way so the character's actions carry the weight. Generating a large batch and reading the names aloud is often faster than brainstorming from scratch — you're reacting to real options instead of staring at a blank page. If you need names that read as intentionally fictional rather than plausible, the fake name generator works well for that too.

    Tips for picking a character name from the list

    • Read it aloud. Names that look fine on the page sometimes trip over the tongue — say a shortlist out loud before committing.
    • Check your cast for collisions. Avoid giving two major characters names that start with the same letter or sound alike, since readers mix them up.
    • Match the register. A gritty crime novel and a cozy small-town romance usually call for different naming styles — generate a large batch and look for the tone that fits.
    • Keep a shortlist. Use "Copy All" to save a batch of twenty or so, then narrow it down over a day instead of deciding on the spot.

    Common questions

    Frequently asked questions

    Will two characters ever get the same name?

    Each generated batch avoids duplicate names within that batch, but running the generator again can repeat names from an earlier batch.

    Can I generate names for a specific gender only?

    Yes — set the gender filter to male or female, or leave it on "both" for a mixed cast.

    Can I use these names in a published book or game?

    Yes. These are common names in random combination, not tied to a real person, so they're generally safe for fiction and commercial projects.

    Is there a limit on how many I can generate?

    You can generate up to 1000 names per click, and run the generator as many times as you like.

    Can I generate just last names for a family or dynasty?

    Yes — set "Name Type" to "Last Only" to get a list of surnames without first names attached.

    What if a name accidentally matches a real public figure?

    With random combination this is rare, but if it happens, simply regenerate or manually swap the surname.

    Is this good for naming background or minor characters quickly?

    Yes — this is exactly the use case it's fastest for. Generate a large batch and assign names to minor characters as you write, without breaking flow.

    Can I generate names that sound period-appropriate for a historical setting?

    The database leans toward names common in the last century, so it works best for contemporary or near-historical settings; for older or highly stylised periods, treat generated names as a starting point to adjust rather than a final answer.
